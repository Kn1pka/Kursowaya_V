    🎮 Memory Game

   > Курсовая работа по дисциплине «Основы программирования»

   Десктопная игра «Memory» (Мемори) — классическая настольная игра на тренировку зрительной памяти с графическим
 интерфейсом Windows Forms.


    🚀 Функционал

   - Три уровня сложности: 4×4, 8×8, 10×10
   - Система учёта статистики: время, ходы, рекорды
   - Пауза с остановкой таймера
   - Асинхронная анимация переворота карт
   - 50 уникальных эмодзи-карточек
   - Модульные тесты (NUnit)

    🛠 Технологии

   - C# (.NET 10)
   - Windows Forms
   - NUnit — модульное тестирование

    📦 Установка и запуск

   ```bash
   git clone https://github.com/твой-логин/Kursowaya_V.git
   cd Kursowaya_V/Kursowaya_V
   dotnet run
 ```

 🧪 Запуск тестов

 ```bash
   cd Kursowaya_V
   dotnet test
 ```
